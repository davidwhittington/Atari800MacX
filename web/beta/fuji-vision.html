Fuji-Vision

Atari 8-bit emulation for Apple Vision Pro. Your favorite 8-bit games and software, floating in your space.

visionOS 2.0+ Apple Vision Pro

Spatial Display

The Atari's display rendered via Metal into your spatial environment, with full control over how it blends with your surroundings.

  • Metal-accelerated rendering in a resizable window
  • Visibility compositor with multiple blend modes
  • Chroma key — make backgrounds transparent
  • Auto-detect background color for seamless keying
  • Edge enhancement for crisp pixel edges
🕶

Game Controller

Bring your own controller. Fuji-Vision supports Bluetooth game controllers for authentic joystick gameplay.

  • Bluetooth controller support via Game Controller framework
  • D-pad and thumbstick input mapping
  • Console key buttons (Start, Select, Option, Reset)
  • On-screen virtual controls as fallback
🎮

Authentic Audio

Full POKEY sound emulation through AVAudioEngine for low-latency spatial audio output.

  • POKEY sound chip emulation
  • Stereo POKEY support
  • Low-latency pull-model audio via AVAudioSourceNode
  • Spatial audio ready for future immersive modes
🎵

Media Management

Load disk images, cartridges, cassettes, and executables through the native visionOS file system.

  • Disk images: ATR, XFD, ATX, PRO, DCM
  • Cartridges: CAR, ROM, BIN (160+ types)
  • Cassettes: CAS format
  • Executables: XEX, COM, EXE
  • Persistent recently-used media tracking
💾

Save States

Save and restore your emulation state at any time with 10 independent save slots.

  • 10 save state slots
  • Persistent across app launches
  • Full machine state capture
  • Settings persistence via @AppStorage
💾
