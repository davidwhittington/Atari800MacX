Fuji-Swift

A lightweight, Swift-native Atari 8-bit emulator for macOS. Streamlined UI, modern codebase, focused feature set.

Planned macOS SwiftUI Native

What is Fuji-Swift?

Fuji-Swift is a planned lightweight variant of the Fuji emulator family. Where Fuji-Foundation carries the full weight of decades of features, Fuji-Swift strips down to the essentials — a clean SwiftUI interface, fast startup, and the core emulation you need for casual 8-bit computing.

Built entirely in Swift with a thin C bridge to the Atari800 core, Fuji-Swift targets users who want a simple, beautiful emulator without the complexity of expansion hardware or advanced debugging tools.

Planned Features

  • Pure SwiftUI interface — native macOS look and feel
  • Metal rendering pipeline (shared with Fuji-Foundation)
  • Disk, cartridge, and executable loading
  • Game controller support via Game Controller framework
  • Save states with Quick Save / Quick Load
  • Lightweight — fast launch, minimal resource usage
  • Drag-and-drop media loading
  • Settings Sync via iCloud

What it won't include

Fuji-Swift intentionally omits advanced features to stay focused:

  • No built-in debugger or CPU monitor
  • No expansion hardware (SIDE2, Ultimate 1MB, VBXE)
  • No cassette tape emulation
  • No hard disk or PCLink emulation
  • No 80-column card support

Status

Fuji-Swift is in the planning stage. The app scaffold exists in the monorepo at apps/fuji-swift/, but active development has not yet begun. Foundation and Vision are the current priorities.
