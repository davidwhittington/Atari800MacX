Atari800MacX

The classic Atari 8-bit emulator for macOS. Cycle-accurate emulation with a modern Metal rendering pipeline.

macOS 13+ Universal Binary Apple Silicon Native
Download Read the Docs

Display & Video

GPU-accelerated rendering through Apple's Metal framework, with full control over how the Atari's display reaches your screen.

  • Metal rendering pipeline — no SDL dependency for display
  • Three width modes: narrow, default, and full overscan
  • Five NTSC artifact color modes
  • 80-column display cards: Bit3 and XEP80
  • Pixel aspect ratio toggle for authentic 4:3 output
◼

Media & Storage

Comprehensive media support covering every storage device the Atari platform ever used.

  • 8 simultaneous disk drives (D1: through D8:)
  • 160+ cartridge bank-switching schemes
  • Cassette tape emulation with motor control
  • Hard disk and PCLink host device emulation
  • ATR, XFD, ATX, CAS, XEX, CAR format support
💾

Sound

Accurate POKEY sound generation with support for dual-POKEY stereo and audio capture.

  • Full POKEY audio emulation
  • Stereo POKEY (dual chip) support
  • Console speaker and SIO motor sounds
  • AIFF audio recording to file
🎵

Machine Variants

25 machine configurations spanning the full Atari 8-bit product line and popular hardware modifications.

  • Atari 400/800 with 8 KB to 48 KB RAM
  • Atari 600XL, 800XL, 1200XL, 65XE, 130XE
  • Atari 320XE and expanded RAM configurations
  • Ultimate 1MB — up to 1024 KB with SIDE2
  • Custom OS ROM and BASIC ROM loading
🖥

Expansion Hardware

Faithful emulation of the most popular expansion devices from the Atari ecosystem.

  • SIDE2 cartridge interface with CompactFlash
  • Ultimate 1MB memory expansion
  • BlackBox and MIO hard disk interfaces
  • VBXE — Video Board XE with 512 KB VRAM
  • R-Time 8 clock cartridge
🔨

Developer Tools

A complete debugging environment for Atari software development and reverse engineering.

  • Built-in 6502 CPU monitor
  • Breakpoints with address and condition support
  • Hardware register viewer (ANTIC, GTIA, POKEY, PIA)
  • Memory browser with hex/ASCII display
  • Instruction-level single stepping
