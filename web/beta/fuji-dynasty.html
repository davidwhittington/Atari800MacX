Fuji-Dynasty

The modular, feature-rich Atari 8-bit emulator. Plugin architecture for expansion hardware, advanced debugging, and community-contributed extensions.

Planned macOS Modular Architecture

What is Fuji-Dynasty?

Fuji-Dynasty is the ambitious, extensible variant of the Fuji emulator family. Where Foundation is a faithful port and Swift is a streamlined experience, Dynasty is designed from the ground up for modularity — every expansion device, every peripheral, every debug tool is a plugin.

Dynasty targets power users, hardware developers, and the Atari homebrew community who need deep control over emulated hardware configurations and the ability to extend the emulator with custom modules.

Planned Architecture

  • Plugin system for expansion hardware modules
  • Hot-swappable device configurations at runtime
  • Swift Package Manager for community plugin distribution
  • Scriptable automation via embedded scripting engine
  • Multi-window workspace — debugger, memory, registers, display

Planned Features

  • All Fuji-Foundation capabilities as baseline
  • Advanced 6502 debugger with conditional breakpoints and watchpoints
  • Memory heatmap visualization
  • ANTIC/GTIA display list explorer
  • POKEY audio waveform monitor
  • Network play — connect emulated machines via TCP
  • Automated testing harness for Atari software
  • Custom hardware device API for community plugins

Status

Fuji-Dynasty is in early concept stage. The app scaffold exists in the monorepo at apps/fuji-dynasty/, but design work on the plugin architecture has not yet begun. This is the longest-horizon project in the Fuji family.
