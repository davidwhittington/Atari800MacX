Getting Started

Your first steps with Atari800MacX on macOS.

What is Atari800MacX?

Atari800MacX is a full-featured Atari 800/XL/XE/5200 emulator for macOS. It provides cycle-accurate emulation of all Atari 8-bit hardware—CPU, ANTIC, GTIA, POKEY, and PIA—along with support for expansion hardware like SIDE2, Ultimate 1MB, BlackBox, MIO, and VBXE. The emulator uses Metal-accelerated rendering, supports up to 8 floppy drives, 5 printer types, 80-column display cards, and includes a built-in CPU debugger.

Note

Atari800MacX includes built-in Altirra OS and BASIC ROMs by Avery Lee. You do not need to supply your own ROM files to get started, although original Atari ROMs can be used if you have them.

First Launch

When you first open Atari800MacX, you will see a blue Atari BASIC screen with the READY prompt. This is the built-in Altirra BASIC ROM running on an emulated Atari XL/XE with 64 KB RAM in NTSC mode.

You can type BASIC commands directly using your Mac keyboard. The Atari keyboard layout is mapped to the Mac keyboard—most keys work as expected, with a few differences noted in the Input & Controllers section.

Tip

If you have original Atari ROM files, you can configure them in Preferences (⌘,) to get the most authentic experience. The built-in Altirra ROMs provide excellent compatibility for most software.

Loading Your First Game

To load a game from a floppy disk image:

  1. Choose Media > Insert Floppy > D1: from the menu bar, or press ⌘1.
  2. Browse to your .atr or .xfd disk image file and select it.
  3. The disk mounts into drive D1: automatically. Most games boot immediately.
  4. If the game doesn't start, try Control > Cold Reset (⇧F5) to reboot with the disk loaded.

You can mount disks in drives D1: through D8: using ⌘1 through ⌘8. Remove all disks with ⌘0.

Tip

For multi-disk games, mount the first disk in D1: and use Media > Rotate Floppies to cycle through disks when prompted, or mount additional disks in D2: through D8:.

Executables (.xex) are loaded directly into memory and run immediately—no disk image required. This is the simplest way to run standalone programs.

Using a Gamepad

Atari800MacX supports game controllers via SDL2 HID. Most USB and Bluetooth controllers are recognized automatically. To verify your controller is detected, look for the Gamepad Identify window.

Keyboard Joystick Mode

If you don't have a gamepad, press ⌘J to enable keyboard joystick mode. The arrow keys control the joystick directions, and you can configure the mapping in Preferences.

Choosing a Machine Type

Atari800MacX emulates three families of Atari hardware, selectable from Control > Atari Machine Type:

FamilyCommon ChoicesNotes
Atari 800800 (48 KB)Original 1979 home computer. No built-in BASIC—requires BASIC cartridge for BASIC programs.
Atari XL/XE800XL (64 KB), 130XE (128 KB)Enhanced 1983+ models with built-in BASIC. Best compatibility with most software. Default.
Atari 52005200 (16 KB)Game console variant with analog joystick. Uses separate 5200 ROM and game cartridges.

Extended memory variants are available for advanced users—320 KB RAMBO, 576 KB, 1088 KB, and Ultimate 1MB configurations. See the User Guide for the full list of 25 machine variants.

Note

Changing the machine type performs a cold reset. Any unsaved work in the emulator will be lost.

Where to Find Games

The Atari 8-bit platform has a rich library of thousands of commercial and homebrew titles. Many classic games and demos are available from community software archives on the web. Search for "Atari 8-bit disk images" or "Atari 800 software archive" to find collections of .atr disk images and .xex executables compatible with Atari800MacX.
