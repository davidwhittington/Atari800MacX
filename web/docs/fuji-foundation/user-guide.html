User Guide

Complete feature reference for Atari800MacX.

Floppy Disks

Atari800MacX supports up to 8 floppy disk drives (D1: through D8:), just like the original Atari hardware with multiple drives daisy-chained via the SIO bus.

Inserting & Removing Disks

ActionMenuShortcut
Insert disk in D1:–D8:Media > Insert Floppy > Dn:⌘1–⌘8
Remove disk from D1:–D8:Media > Remove Floppy > Dn:⌘⌃1–⌘⌃8
Remove all disksMedia > Remove Floppy > All⌘0
Rotate floppiesMedia > Rotate Floppies—

Rotate Floppies shifts all mounted disks forward by one drive (D1: → D2:, D2: → D3:, etc.), wrapping the last disk back to D1:. This is useful for multi-disk games that prompt you to swap disks.

Disk Management Window

Open Media > Disk Management (⌘D) for an overview of all 8 drives. From this window you can mount, unmount, and write-protect individual drives.

Creating & Editing Disk Images

  • New Floppy Image (⌘N) — creates a blank formatted disk image
  • Edit an .atr Disk Image (⌘E) — file-level editor for viewing and modifying disk contents
  • Sector Edit an .atr Image (⌘⇧E) — low-level sector editor for direct hex editing

Disk Image Conversions

Convert between disk image formats via Media > Disk Image Conversions:

ConversionDirection

DCM ↔ ATRDiskComm compressed to/from ATR
SCP ↔ ATRSuperCard Pro to/from ATR

Cartridges

Standard Cartridges

ActionShortcut
Insert Cartridge⌘O
Remove Cartridge⌘⇧O
Insert BASIC Cartridge⌘B

Atari800MacX supports over 160 cartridge types covering every known bank-switching scheme—from simple 8 KB cartridges to 128 MB The!Cart flash cartridges. When inserting a .car file, the cartridge type is detected automatically from the header. For raw .rom or .bin files, you may be prompted to select the correct type.

Flash & RAM Cartridges

Under Media > Handle Special Cartridge > Insert Blank Flash/RAM Cartridge, you can create blank writable cartridges:

  • MaxFlash — 128 KB (1 Mbit) or 1 MB (8 Mbit)
  • The!Cart — 32 MB, 64 MB, or 128 MB
  • MegaCart — 512 KB, 1 MB, or 4 MB
  • SiC! — 128 KB, 256 KB, 512 KB, or SiC!+
  • J(atari) — 128 KB to 1024 KB
  • Ram-Cart — 64 KB through 32 MB
  • SiDiCar — 32 KB
  • DCart

Flash cartridge state can be saved with Media > Handle Special Cartridge > Save Flash/RAM Cartridge.

Cartridge Conversions

  • Create .car From ROM File — wraps a raw ROM image in a CAR header with the correct type
  • Create ROM File From .car — strips the CAR header to produce a raw ROM image

Cassettes

Cassette tape emulation supports .cas tape images:

RecordMedia > Cassette Record

Tip

To boot from a cassette, hold Start during a cold reset. The Atari will attempt to load from the cassette instead of disk. With the SIO patch enabled, cassette loading is nearly instantaneous.

Executables

Load Atari binary executables directly with Media > Load Executable File (⌘R). Supported formats include .xex, .com, .exe, and .obx. The file is loaded into memory and executed immediately—no disk drive required.

Hard Disks

H: Device (Host Directories)

The H: device maps up to 4 directories on your Mac to Atari device paths (H1: through H4:). Software on the Atari can read and write files in these host directories as if they were connected hard drives. Configure the mount points and read-only toggle in Preferences.

PCLink

PCLink provides an alternative host directory interface with support for 4 directories. It uses the SIO bus protocol for communication, offering compatibility with software that expects PCLink hardware.

Creating Hard Disk Images

Use Media > New Hard Disk Image to create a blank hard disk image file for use with expansion hardware like SIDE2 or BlackBox.

80-Column Display

Atari800MacX emulates three 80-column display expansion cards that provide 80-character-wide text output:

CardResolutionColors

Select a card from Display > 80 Column Card Selection. The XEP80 can be connected to either Joystick Port 1 or Port 2.

Display Switching

  • Display 80 Column Screen (⌘⇧X) — toggles between the standard 40-column Atari display and the 80-column card output
  • 80 Column Display Autoswitch — automatically switches to the 80-column display when the card is active

Sound & Audio

SettingMenu / ShortcutDescription
Enable SoundSound > Enable SoundMaster audio toggle
Stereo POKEYSound > Stereo Sound (⌘U)Dual-POKEY stereo output for compatible software
Volume Up⌘]Increase volume
Volume Down⌘[Decrease volume
Sound RecordingSound > Start Sound Recording (⌘T)Record audio output to AIFF file

Sound is generated by the emulated POKEY chip (4 channels). The emulator supports console key click sounds, SIO bus sounds, and disk drive motor sounds for a more authentic experience. These can be configured in Preferences.

Note

Stereo POKEY mode enables a second POKEY chip for stereo output. Only software specifically written for dual-POKEY hardware (certain demos and music programs) will produce stereo sound. Most games use a single (mono) POKEY.

Memory Expansions

Several memory expansion schemes are emulated for the Atari 800:

  • Axlon — bank-switched RAM expansion using I/O port at $CFFF
  • Mosaic — alternative bank-switched RAM expansion

For XL/XE models, extended memory is accessed via PORTB ($D301) banking. The RAMBO and Compy Shop schemes differ in how they map the additional banks.

Expansion Hardware

SIDE2 Cartridge

The SIDE2 is a CompactFlash-based storage cartridge. Insert it with Media > Handle Special Cartridge > Insert SIDE2 Cartridge (⌘I).

  • CompactFlash — attach/remove a hard disk image as the CompactFlash device
  • SDX / Loader Switch — toggle between SpartaDOS X boot mode and the SIDE2 loader menu
  • SIDE2 Button — press the physical button (triggers menu/reset)
  • ROM Management — change or save the SIDE2 flash ROM image

Ultimate 1MB

The Ultimate 1MB expansion provides 1 MB of flash ROM with multiple OS/BASIC/cartridge slots, PBI BIOS support, and VBXE integration. Select an Ultimate 1MB machine variant from the Control menu, then manage the ROM image via Media > Handle Ultimate1mb ROM Images.

PBI Expansion Devices

Select a PBI (Parallel Bus Interface) device from Control > PBI Expansion Type:

DeviceDescription
BlackBoxSCSI hard disk interface with parallel printer port and RS-232 serial port
MIO (Multi I/O)SCSI hard disk and RAM disk interface

VBXE (Video Board XE)

VBXE is a video expansion that adds an overlay graphics layer with 256 colors from a palette of 21-bit color, hardware blitter, and extended display list (XDL). VBXE is activated through the Ultimate 1MB machine variants. See the Technical Reference for register details.

Input & Controllers

Keyboard Joystick Mode

Toggle with ⌘J. When enabled, the keyboard can be used as a joystick. The arrow key behavior is controlled by the mode selected under Control > Arrow Keys.

CX85 Numeric Keypad

Enable via Control > Enable CX85 Keypad. Maps the Mac numeric keypad to the Atari CX85 numeric keypad peripheral, used by some business and productivity software.

XEGS Keyboard

Enable via Control > XEGS Keyboard Attached. Simulates the detachable keyboard of the Atari XEGS game system.

Gamepad Support

USB and Bluetooth game controllers are supported via SDL2 HID. Controllers are detected automatically. The Gamepad Identify window shows the currently connected controller and its mapping.

Mouse Modes

Use Display > Grab Mouse to capture the mouse pointer for emulating analog input devices:

  • Paddles — horizontal mouse movement controls paddle position
  • Light pen — mouse position maps to screen coordinates
  • Trackball — mouse movement maps to trackball input

Save & Load State

ActionShortcutDescription
Save State⌘SSave the complete emulator state to an .a8s file
Load State⌘LRestore a previously saved state
Save Configuration⌘⇧SSave emulator settings to an .a8c file
Load Configuration⌘⇧LLoad emulator settings from an .a8c file

State files (.a8s) capture the entire emulator state, including CPU registers, memory contents, mounted disks, and cartridge state. Configuration files (.a8c) save only the emulator settings (machine type, display options, ROM paths, etc.) without the runtime state.

Tip

Save states are a quick way to bookmark your progress in a game. Configuration files are useful for maintaining different setups for different types of software (e.g., one config for gaming, another for development).

Emulation Patches

Atari800MacX can patch the emulated OS for faster I/O and network features:

  • SIO Patch — bypasses the serial I/O routines to load disks and cassettes at maximum speed instead of simulating real SIO bus timing
  • H: Patch — enables the H: device for host directory access
  • D: Patch — patches the disk handler
  • P: Patch — patches the printer handler
  • R: Patch — patches the RS-232 handler

Network Features

  • NetSIO — enables network SIO communication (toggle via Control > Enable NetSIO)
  • FujiNet — FujiNet network device emulation

Preferences

Open Preferences with ⌘,. The preferences panel provides access to all emulator settings organized by category.

Key Preference Areas

  • Video — TV mode (NTSC/PAL), artifacting, color palette adjustment, LED indicators
  • Audio — sound enable, volume, stereo POKEY, console/SIO/drive sounds
  • Machine — machine type, memory configuration, OS patches
  • Input — joystick configuration, arrow key mode, CX85 keypad, gamepad settings
  • Peripherals — H: device directories, PCLink directories, printer configuration, 80-column cards
  • Paths — ROM file locations (OS, BASIC, 5200), default disk/cartridge/cassette directories

The new SwiftUI preferences panel provides a modern interface for the most common settings. The legacy ObjC preferences panel remains available for advanced settings not yet migrated.
