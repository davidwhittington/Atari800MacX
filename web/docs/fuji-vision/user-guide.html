User Guide

Complete feature reference for Fuji-Vision.

Media Management

Executable.xex .com .exe .obxAtari binary load files
Cassette.casCassette tape images

Mounting Media

Use the toolbar buttons to mount media into the emulator:

  • Disks — Insert a disk image into Drive D1: or D2:. Each drive has its own Insert and Eject options.
  • Cartridge — Insert or remove a cartridge ROM.
  • More — Load an executable file or mount/eject a cassette tape.

When you select a file, it is copied into the app's sandboxed Documents/Media/ folder. The original file on your device is never modified.

Media Status Ornament

When media is mounted, a glass-style status bar appears along the bottom of the emulator window showing the filenames of all currently loaded media. This ornament automatically hides when no media is mounted.

Disk LED Indicator

A small circle appears in the top-right corner of the emulator display during disk access:

  • Green — disk read operation
  • Red — disk write operation

Media Persistence

Mounted media is remembered across app launches. When you reopen Fuji-Vision, the same disks, cartridges, and cassettes will be loaded automatically.

Display Settings

Open Settings (gear icon) to access display options.

TV Mode

Choose between NTSC and PAL video standards. NTSC runs at 60 Hz with North American color palettes. PAL runs at 50 Hz with European colors. Most North American software expects NTSC.

Artifacting

The Atari's NTSC video output produced color artifacts from high-resolution pixel patterns. Fuji-Vision can simulate this effect:

ModeDescription
NoneClean pixel output, no artifact simulation
NTSC OldClassic NTSC artifact coloring
NTSC NewImproved artifact algorithm
NTSC New Hi-FiHighest-quality NTSC simulation
Tip

Some games (like Karateka or Drol) were designed with NTSC artifacting in mind and will look correct only with an artifact mode enabled.

Bilinear Filter

When enabled, applies a smooth (bilinear) filter to the display instead of sharp nearest-neighbor pixels. This softens the image, resembling how the picture looked on an analog TV.

CRT Scanlines

Simulates the visible horizontal scan lines of a CRT television. When enabled, a Scanline Intensity slider appears (range 50%–100%) controlling how prominent the dark lines are. Higher values make the lines less visible.

Visibility & Transparency

Fuji-Vision's visibility compositor lets you control how opaque the emulator window appears in your spatial environment, and optionally key out background colors so the real world shows through.

Visibility Modes

ModeOpacityDescription
Solid100%Fully opaque, traditional display
Dim60%Semi-transparent, see-through background
Ghost20%Mostly transparent, faint overlay

Mode transitions are animated over 0.3 seconds for visual comfort.

Peek Gesture

Press and hold anywhere on the emulator display to temporarily reduce opacity to 15%. Release to return to your selected visibility mode. This lets you quickly glance at your surroundings without changing settings.

Chroma Key

When enabled, the chroma key removes a specific color from the display and makes those pixels transparent—similar to a green screen effect. This is useful for making the Atari's solid background color disappear so only the game graphics float in space.

SettingDescription
Chroma KeyMaster toggle for color keying
ThresholdHow closely a pixel must match the key color to be removed (0.0–0.5)
Soft EdgeWidth of the gradual blend between keyed and non-keyed areas (0.0–0.3)
Invert KeyReverses the key: keeps only pixels matching the key color, removes everything else

Auto-Detect Background

When enabled, the compositor automatically samples the border pixels of the display every half-second to detect the dominant background color. Once a stable color is detected (consistent across three consecutive samples), it is automatically set as the chroma key color.

  • Sensitivity — Controls how tolerant the detection is to color variation (0.0–1.0). Higher values require a more exact match.
  • Lock Detected Color — Freezes the currently detected key color so it won't change as screen content changes.

Edge Enhancement

Sharpens the edges of game graphics when chroma keying is active, reducing halo artifacts at the boundary between keyed and non-keyed regions.

Speed Control

SettingDefaultDescription
Speed LimitOnCaps emulation at the selected speed. Disable to run as fast as possible.
Speed100%Emulation speed multiplier: 50%, 75%, 100%, 125% … up to 400%, in 25% steps
Tip

Reducing speed to 50% can help with games that are too fast. Increasing to 200%–400% is useful for speeding through loading screens or slow simulations.

Save States

Fuji-Vision provides 10 save slots that capture the complete emulator state, letting you save and resume from any point in a game.

Using Save States

  1. Tap the save icon in the toolbar to open the Save States panel.
  2. Each slot shows its number and the date/time of the last save (or "Empty" if unused).
  3. Tap Save to capture the current state into that slot.
  4. Tap Load to restore a previously saved state.
  5. Tap the trash icon to delete a saved state.

Storage

Save state files use the .a8s extension and are stored in the app's Documents/SaveStates/ directory. Files are named state_1.a8s through state_10.a8s.

Input

Game Controller

See the Getting Started controller section for the full button mapping table.

Key details:

  • Both the D-pad and left analog stick control the joystick (8-way directional, including diagonals).
  • Dead zone of 30% on the analog stick prevents drift.
  • Only the first connected controller is used.
  • If a controller disconnects mid-game, the joystick resets to centered with fire released.

On-Screen Controls

See the Getting Started on-screen controls section for layout details.

Console Keys

The Atari's front-panel console keys serve specific functions:

KeyFunction
StartStarts a game or confirms a menu selection. Mapped to controller B.
SelectCycles through game options (number of players, difficulty, etc.). Mapped to controller X.
OptionSelects sub-options in some games. On XL/XE, holding Option during boot disables BASIC. Mapped to controller Y.
ResetWarm resets the emulator (soft reboot without power-cycling). Mapped to controller Menu.

Keyboard Shortcuts (Shoulder Buttons)

The left and right shoulder buttons on a game controller provide quick access to commonly needed keys:

  • Left Shoulder → Space (useful for advancing text or pausing in many games)
  • Right Shoulder → Return (confirms input in BASIC and many programs)

App Lifecycle

Background & Foreground

When Fuji-Vision moves to the background (e.g., you switch to another app), emulation pauses automatically. When you return to Fuji-Vision, emulation resumes—unless you had manually paused it before leaving, in which case it stays paused.

Settings Persistence

All settings are saved automatically via @AppStorage and persist across app launches. There is no "save settings" button—changes take effect immediately and are remembered.

Pause & Resume

Tap the Pause/Resume button in the toolbar to manually pause or resume emulation at any time. The pause state is indicated by the toolbar button icon changing between pause and play symbols.
