Getting Started

Your first steps with Fuji-Vision on Apple Vision Pro.

What is Fuji-Vision?

Fuji-Vision is an Atari 800/XL/XE/5200 emulator designed from the ground up for Apple Vision Pro. It brings the full 8-bit Atari experience into your spatial computing environment—complete with Metal-accelerated rendering, spatial audio, game controller support, and a visibility compositor that lets the Atari display float transparently over your real-world surroundings.

Requirements

RequirementDetails

Tip

A game controller is highly recommended for the best experience. While on-screen touch controls are available, a physical controller is much more comfortable for extended play.

First Launch

When you first open Fuji-Vision, you will see a blue Atari BASIC screen with the READY prompt. This is the built-in Altirra BASIC ROM—no external ROM files are needed to get started.

The emulator defaults to the Atari XL/XE machine type with NTSC video mode. You can type BASIC commands directly using the on-screen keyboard or a connected Bluetooth keyboard.

Note

Fuji-Vision includes built-in Altirra OS and BASIC ROMs. You do not need to supply your own ROM files, although original Atari ROMs can be used if you have them.

Loading Your First Game

To load a game from a disk image:

  1. Tap the Disks button in the toolbar (disc icon).
  2. Select Insert Disk under Drive D1:.
  3. Browse to your .atr or .xfd disk image file and select it.
  4. The disk will mount automatically. Many games boot immediately; others may require you to press the Reset button or power-cycle the emulated machine.

You can also load other media types from the toolbar:

  • Cartridge (chip icon) — insert .car, .rom, or .bin cartridge files
  • More (ellipsis icon) — load executables (.xex) or mount cassettes (.cas)
Tip

Files you import are copied into the app's sandboxed Documents folder. Your originals are never modified.

Using a Game Controller

Pair a Bluetooth game controller through Settings > Bluetooth on your Apple Vision Pro. Fuji-Vision automatically detects the first connected controller.

Note

The analog stick has a dead zone of 30%. Small stick movements near center are ignored to prevent drift.

Using On-Screen Controls

If you don't have a game controller, Fuji-Vision provides on-screen touch controls:

  • D-pad (left side) — four directional buttons for joystick movement
  • Fire (right side) — large red circle button
  • Console keys (top center, hidden by default) — tap the keyboard icon in the top-right corner to reveal Start, Select, Option, and Reset

Console key presses are momentary—the key is held for a brief instant then released automatically, matching how the original Atari console keys work.

Changing the Machine Type

Fuji-Vision emulates three Atari machine families:

MachineDescription
Atari 800Original 1979 home computer (48K RAM, no built-in BASIC)
Atari XL/XEEnhanced 1983+ models (64K RAM, built-in BASIC)
Atari 5200Game console variant (analog joystick)

To switch machines, tap the Machine button in the toolbar (desktop computer icon) and pick your target. The emulator will reset to the new machine type immediately.

You can also change the machine type from Settings > Machine.

Quick Settings

Tap the gear icon in the toolbar to open Settings. The most commonly adjusted options:

SettingWhat it does
TV ModeSwitch between NTSC (60 Hz, North American colors) and PAL (50 Hz, European colors)
Sound EnabledToggle audio output on or off
Speed LimitWhen enabled, caps emulation at real-time speed. Disable to run as fast as possible.
SpeedAdjust emulation speed from 50% to 400% (when speed limit is on)

Where to Find Games

The Atari 8-bit platform has a rich library of thousands of commercial and homebrew titles. Many classic games and demos are available from community software archives on the web. Search for "Atari 8-bit disk images" or "Atari 800 software archive" to find collections of .atr disk images compatible with Fuji-Vision.
