# fuji-dynasty

The feature-rich, modular version of fuji-foundation. Additional capabilities are added
as opt-in modules on top of the core.

## Relation to fuji-foundation

fuji-dynasty is derived from fuji-foundation (Atari800MacOS core) and extends it with
optional modules. Each module is self-contained and can be enabled/disabled independently.
